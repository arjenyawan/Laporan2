1. Mengapa kami menggunakan CSS dan JavaScript dalam Pemograman Web?

CSS adalah salah satu dari dua teknologi penting dalam pengembangan web. CSS digunakan untuk menata dan menata halaman web, yang memberikan estetika sekaligus keterbacaan konten. CSS memungkinkan pengembang web untuk mengontrol elemen-elemen seperti warna, margin, padding, dan tata letak secara konsisten, sehingga meningkatkan pengalaman pengguna. Sementara itu, JavaScript adalah bahasa pemrograman yang memungkinkan pengembang untuk mengontrol dan memanipulasi elemen-elemen pada halaman web secara interaktif. Misalnya, JavaScript dapat digunakan untuk membuat popup, membuat animasi, dan memvalidasi formulir. Tentu saja, ini hanyalah beberapa contoh tentang apa yang dapat dilakukan gabungan CSS dan JS, namun, keduanya seharusnya dapat memberikan gambaran umum. Gabungan keduanya membuat halaman web lebih menarik, efisien, dan ramah pengguna. Sementara CSS memungkinkan struktur menjadi rapi dan seragam, JavaScript menambahkan logika dan interaktivitas ke sisi klien yang berarti di dalam browser pengguna dan tidak mengharuskan halaman web dimuat ulang agar perubahan diterapkan yang meringankan beban di server dan memungkinkan waktu respons yang lebih cepat.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS dan JavaScript dapat diintegrasikan ke dalam HTML dengan tiga cara: inline, internal, dan eksternal. Untuk CSS:
• Inline CSS diterapkan langsung pada elemen HTML menggunakan atribut style. Sebaiknya hindari penggunaan inline ini kecuali untuk gaya yang sangat sederhana, karena dapat membuat HTML menjadi sulit dibaca.
• Internal CSS menggunakan tag style di dalam file HTML. Ini berguna untuk gaya yang spesifik pada satu halaman, tetapi tidak disarankan untuk proyek yang lebih besar.
• Eksternal CSS memisahkan file CSS dari HTML dengan menggunakan file .css. Metode eksternal lebih efisien karena memungkinkan pengembang untuk mengelola semua gaya halaman web di satu tempat, sehingga lebih mudah untuk digunakan kembali dan diatur (W3Schools, 2022).
JavaScript juga memiliki tiga metode integrasi yang serupa: • Inline JavaScript menambahkan skrip langsung pada elemen HTML dengan atribut seperti onclick, onchange, dan lainnya. Sama seperti CSS, metode ini jarang direkomendasikan.
• Internal JavaScript ditempatkan dalam tag script di bagian head atau body HTML. Ini cocok untuk kode yang spesifik pada satu halaman.
• Eksternal JavaScript memisahkan kode ke dalam file .js, yang dapat dihubungkan melalui tag script src="file.js". Metode ini sangat disarankan karena dapat digunakan kembali di banyak halaman web, mempercepat waktu loading, dan meningkatkan pengelolaan kode (MDN Web Docs, 2021).

4. Jelaskan apa saja keuntungan menggunakan JavaScipt dan CSS?

Bootstrap adalah framework CSS yang sangat terkenal karena menyediakan berbagai komponen desain yang siap digunakan dan memudahkan pengembangan antarmuka pengguna (UI). Beberapa keuntungan dari Bootstrap antara lain:
• Kecepatan Pengembangan: Bootstrap menawarkan berbagai komponen siap pakai seperti tombol, formulir, grid, dan navbar yang bisa langsung digunakan tanpa perlu menulis kode dari awal, sehingga mempercepat proses pembuatan prototipe (GetBootstrap, 2022).
• Desain Responsif: Bootstrap mendukung desain responsif yang secara otomatis menyesuaikan elemen UI pada berbagai ukuran layar, memberikan pengalaman optimal di perangkat mobile. Dukungan Komunitas: Dengan komunitas pengguna yang besar dan dokumentasi yang lengkap, Bootstrap menyediakan banyak tutorial, tema, dan ekstensi. Ini memudahkan pemula untuk memahami dan memodifikasi tampilan yang diinginkan tanpa mengalami banyak kesulitan (Bootstrap Community, 2022).
